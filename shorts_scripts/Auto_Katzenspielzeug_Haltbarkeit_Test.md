## Scripts

Skeptischer Hook → Krallen/Beiß-Test → Proof → CTA.

### Structure (40s)
- Hook: „Mein Kater zerstört alles…“ (0–3s)
- Reveal: Spielzeug in Action, erste Attacke (3–8s)
- Proof: Nahaufnahme Krallen/beißen, Stoff hält (8–22s)
- Payoff: Katze spielt weiter → „nicht nach 5 Minuten kaputt“ (22–32s)
- CTA: „Jetzt ausprobieren“ (32–40s)

### Visual Strategy
| Seconds | Visual | Purpose |
|:--|:--|:--|
| 00-03 | Close-up: Krallen in Stoff | Pattern interrupt |
| 03-08 | Toy bewegt sich, Katze springt | Reveal |
| 08-16 | Detail: Krallen, Pfoten, Stoff/Naht | Proof durability |
| 16-22 | Wiederholung: 2–3 harte „Treffer“ | Trust |
| 22-32 | Weitwinkel: Katze bleibt dran | Payoff |
| 32-40 | Overlay CTA + Produkt im Bild | Conversion |
