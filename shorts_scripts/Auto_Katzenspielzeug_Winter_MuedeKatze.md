## Scripts

„Faule Katze“-Hook → Aktivierung durch Bewegung/Sound → Proof → CTA.

### Structure (36s)
- Hook: Katze schläft / „zu faul zum Spielen?“ (0–3s)
- Reveal: Spielzeug an, erster Blickkontakt (3–8s)
- Proof: mehrere Sprünge / Pfoten / „Jagdinstinkt“ (8–24s)
- Payoff: Katze spielt allein → Owner zufrieden (24–30s)
- CTA: „Probier’s aus“ (30–36s)

### Visual Strategy
| Seconds | Visual | Purpose |
|:--|:--|:--|
| 00-03 | Katze schläft auf Sofa | Relatable pain point |
| 03-08 | Spielzeug startet, Ohren gehen hoch | Trigger moment |
| 08-16 | Close-up: Pfoten „fangen“ unter Stoff | Proof |
| 16-24 | Wiederholung: 2–3 schnelle Pounces | Retention |
| 24-30 | Weitwinkel: Katze beschäftigt, Room ruhig | Payoff |
| 30-36 | Overlay „Probier’s aus ⬇️“ | CTA |
