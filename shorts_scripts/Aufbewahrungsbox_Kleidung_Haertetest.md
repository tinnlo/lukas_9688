## Scripts

Härtetest-Hook (Human Fit) → Kapazität → Reißverschluss/Griffe → Vorher/Nachher → CTA.

### Structure (40s)
- Hook: „Passt da wirklich ’ne Person rein?“ (0–3s)
- Proof 1: Human-Fit/Lift-Test (3–8s)
- Proof 2: Winterdecken/Jacken rein, zip zu (8–22s)
- Payoff: Schrank „vorher/nachher“ ordentlich (22–32s)
- CTA: „Ordnung holen“ (32–40s)

### Visual Strategy
| Seconds | Visual | Purpose |
|:--|:--|:--|
| 00-03 | Person sitzt/steht in Box | Pattern interrupt |
| 03-08 | Griff-Test (anheben/ziehen) | Trust |
| 08-16 | Zeitraffer: Winterzeug rein | Capacity proof |
| 16-22 | Close-up: Reißverschluss + Griffe | Quality cue |
| 22-32 | Vorher/Nachher Kleiderschrank | Payoff |
| 32-40 | Textoverlay CTA + Stack-Shot | Conversion |
