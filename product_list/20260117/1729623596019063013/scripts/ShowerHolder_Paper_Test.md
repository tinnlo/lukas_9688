## Scripts

**Concept**: "The Paper Test" - classic waterproof validation using paper inside case. Addresses #1 fear: "Will my phone get wet?" Trust-building through extreme proof.

### Structure (30s)
- **Hook (0-3s):** Skepticism - "Really waterproof?"
- **Test Setup (3-10s):** Paper inside case
- **Water Test (10-22s):** Extreme water exposure
- **Reveal (22-27s):** Paper still dry
- **CTA (27-30s):** Trust earned

### Visual Strategy

| Seconds | Visual | Purpose |
|:--------|:-------|:--------|
| 00-03 | Creator with skeptical face, text: "Wirklich spritzwassergeschützt?" | DOUBT voice |
| 03-06 | Placing white paper inside waterproof case instead of phone | TEST SETUP - visual indicator |
| 06-10 | Sealing case shut, showing all edges secured | THOROUGHNESS |
| 10-13 | Holding under running shower stream directly | STRESS TEST begins |
| 13-16 | Submerging entire case in sink/bathtub water | EXTREME CONDITIONS |
| 16-19 | Shaking case underwater, bubbles visible | AGITATION test |
| 19-22 | Wiping off exterior, building suspense | ANTICIPATION |
| 22-25 | Opening case slowly - paper is COMPLETELY DRY | VICTORY MOMENT |
| 25-27 | Comparing wet exterior vs. bone-dry interior paper | PROOF contrast |
| 27-30 | Text: "Dein iPhone ist sicher!" + product + cart | TRUST CTA |
