## Scripts

The "Habit Challenge" pattern targeting manual brush users. Opens with immediate audience qualification. Appeals to Germans who value efficiency and health optimization. Uses Google search call-out to build confidence in electric brushing benefits.

### Structure (34s)
- Hook: Challenge current habit (0-3s)
- Problem: Manual brushing limitations (3-8s)
- Solution: Electric brush benefits (8-18s)
- Feature Tour: Timer + modes + USB-C (18-28s)
- CTA: Upgrade call (28-34s)

### Visual Strategy
| Seconds | Visual | Purpose |
|:--------|:-------|:--------|
| 00-03 | Holding dirty old manual toothbrush vs shiny new electric one side-by-side | Hook: Immediate visual contrast |
| 03-08 | Hand-brushing motion (slow, inefficient) vs electric brush vibration (fast, thorough) | Problem: Manual limitations clear |
| 08-12 | Text overlay: "Google: electric vs manual toothbrush benefits" | Trust: Confident in science backing |
| 12-16 | Close-up of bristles moving in mouth (or on hand demo) showing vibration coverage | Education: How it works better |
| 16-20 | Timer demonstration: Brush lights up/pulses every 30 seconds for quadrant change | Feature: Intelligent technology |
| 20-24 | Mode buttons: Clean, Sensitive, Whitening indicators lighting up | Versatility: Personalized care |
| 24-28 | Plugging USB-C cable into handle, modern charging visual | Convenience: No weird charging bases |
| 28-34 | Before/after teeth comparison (or smile reveal), holding brush proudly | Result: Visual payoff + CTA |
