## Scripts

**Concept**: "Active Lifestyle Essential" - target runners/joggers who need to control music/track workout in cold weather. Multi-sport positioning: cycling, running, hiking.

### Structure (28s)
- **Hook (0-3s):** Winter fitness challenge
- **Problem (3-8s):** Bare hands = pain
- **Solution (8-22s):** Gloves enable active lifestyle
- **CTA (22-28s):** Fitness essential

### Visual Strategy

| Seconds | Visual | Purpose |
|:--------|:-------|:--------|
| 00-03 | Runner in winter gear, breath visible, determined face | WINTER ATHLETE identification |
| 03-05 | Close-up: red frozen bare hands, painful expression | PAIN POINT visualization |
| 05-08 | Can't change music on phone, fumbling with frozen fingers | FUNCTIONALITY LOSS |
| 08-11 | Putting on gloves mid-run, instant relief expression | SOLUTION introduction |
| 11-14 | Swiping through Spotify, adjusting volume smoothly while running | TOUCHSCREEN in action |
| 14-17 | Checking fitness tracker app, pausing/starting workout | MULTIFUNCTIONALITY |
| 17-20 | Running through light rain, hands dry and warm | WEATHER RESISTANCE |
| 20-22 | Grip quality: holding water bottle securely with gloved hands | PRACTICAL GRIP |
| 22-25 | Multiple sports montage: cycling, hiking, walking dog | VERSATILITY appeal |
| 25-28 | Text: "Dein Winter-Trainingspartner" + product | ESSENTIAL CTA |
