## Scripts

A skeptic/proof angle that addresses the common objection: "Portable blenders are weak toys that break after one use." This script uses hard evidence (ice cube test + battery display) to build trust with German viewers who value quality and durability.

### Structure (38s)
- Hook: Address skeptic comment directly (0-3s)
- Challenge Setup: Show the "stress test" ingredients (3-7s)
- Demo: Blending action with ice cubes + battery visible (7-25s)
- Proof: Smooth result + battery still high (25-33s)
- CTA: Point to shop link (33-38s)

### Visual Strategy
| Seconds | Visual | Purpose |
|:--------|:-------|:--------|
| 00-03 | Screenshot of negative comment: "Breaks after 1 day" with creator looking annoyed | Hook: Relatable objection creates curiosity |
| 03-07 | Hands dropping LARGE ice cubes + frozen berries into transparent cup | Build tension: "Can it really handle this?" |
| 07-10 | Close-up: LED display showing battery at 95% before starting | Trust: Tech credibility with real data |
| 10-18 | Upside-down blending action, close-up of vortex inside cup showing ice crushing | Proof: Visual satisfaction of power |
| 18-22 | Shake technique (as per manual) - dynamic movement | Education: Proper usage |
| 22-25 | Stop blending, show smooth texture with spoon | Result: Appetizing payoff |
| 25-28 | LED display still showing 93% battery | Trust: "Lasts forever" claim validated |
| 28-33 | Pour into glass, perfect creamy consistency | Sensory: Make viewer thirsty |
| 33-38 | Point to bottom of screen, holding bottle to camera | Conversion: Clear CTA |
