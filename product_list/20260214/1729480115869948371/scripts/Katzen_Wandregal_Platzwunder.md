## Scripts

German city apartment struggle: cluttered floors, oversized cat trees eating up valuable living space. This script attacks the pain point directly (Type 2 hook: "Jeden Tag das gleiche Problem"), then presents wall-mounted solution as space hack. Target audience: city dwellers with 50qm apartments who need every square meter.

### Structure (45s)
- Hook: POV walking through cluttered apartment, stumbling over cat tree (0-3s)
- Problem: Show cramped floor space, contrast with empty wall (3-8s)
- Solution: Install wall hammock system, show mounting (8-20s)
- Benefits: Vacuum robot gliding freely, cat happy above, clean aesthetic (20-35s)
- CTA: "Wandparkour statt Platzverschwendung. Link ist unten." (35-45s)
