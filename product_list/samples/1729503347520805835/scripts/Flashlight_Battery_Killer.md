## Scripts

**Concept**: The "Battery Killer" - attack the recurring cost and annoyance of traditional flashlights that eat through AA batteries. Position USB rechargeability as the smart, economical choice. Target budget-conscious homeowners and outdoor enthusiasts tired of battery expenses.

### Structure (32s)

- **Hook (0-3s):** Sweeping AA batteries into trash - "Stop buying batteries!"
- **Problem (3-10s):** Show dim flickering old flashlight frustration
- **Solution (10-22s):** USB rechargeable demo + 4-core LED brightness reveal
- **Benefits (22-28s):** Versatility showcase (main beam + COB side light)
- **CTA (28-32s):** Value proposition - one-time purchase, lifetime use

### Visual Strategy

| Seconds | Visual | Purpose |
|:--------|:-------|:--------|
| 00-03 | Dramatic sweep: pile of dead AA batteries being dumped into trash bin with sound effect | PAIN POINT AGITATION - visualizes wasted money |
| 03-06 | Old yellow-beam flashlight flickering weakly, then dying completely in hand | FRUSTRATION establishment - the "before" state |
| 06-10 | Text overlay: "AA Batterien: jeden Monat wieder!" | RATIONAL JUSTIFICATION - quantify the problem |
| 10-13 | Close-up: USB cable plugging into flashlight port, blue battery indicator lights up | SOLUTION REVEAL - modern rechargeability |
| 13-16 | Pitch black room → CLICK sound → 4-core LED BLASTS white light (extreme contrast) | BRIGHTNESS PROOF - show power difference |
| 16-20 | Split screen: left = phone flashlight (pathetic), right = this 4-core LED (daylight) | COMPARISON validation - better than common alternative |
| 20-23 | Click button: side yellow COB panel lights up, user reading a book comfortably | VERSATILITY - 2-in-1 value (spotlight + floodlight) |
| 23-26 | Rapid mode cycling: High → Low → Strobe → COB in 2 seconds | FEATURE DENSITY overwhelms doubt |
| 26-28 | Blue battery indicator close-up showing 4 bars (100% charged) | TRUST ELEMENT - smart battery management |
| 28-32 | Side-by-side: Left = pile of batteries + "teuer" text, Right = USB flashlight + "keine Folgekosten" | FINAL VALUE PROPOSITION with CTA overlay |
