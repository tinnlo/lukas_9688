## Scripts

The "Box Reveal" pattern building curiosity and "insider" status. Raw, authentic "friend-to-friend" camera angle. Per video synthesis, the most successful video used this approach for Arabic diaspora - adapted here for broader German audience with "TikTok secret tip" framing.

### Structure (38s)
- Hook: Package arrived announcement (0-3s)
- Build-up: Box tapping ASMR (3-6s)
- Reveal: Fast unboxing (6-14s)
- Discovery: Counting contents excited (14-26s)
- Logic: Shipping hack tip (26-34s)
- CTA: Share discovery (34-38s)

### Visual Strategy
| Seconds | Visual | Purpose |
|:--------|:-------|:--------|
| 00-03 | Package on desk/kitchen table, excited creator pointing at it | Hook: "Package arrived" triggers curiosity |
| 03-06 | Tapping fingernails on plastic box creating ASMR sound | Build-up: Sensory anticipation |
| 06-10 | Ripping box open quickly, pulling out items fast | Energy: Authentic excitement |
| 10-14 | Pulling out toothbrush handle, examining it with surprise | Discovery: "Better than expected" |
| 14-18 | Dumping brush heads, starting to count excitedly | Abundance: Visual value |
| 18-22 | Holding up charging cable, showing USB-C plug | Modern: Quality signal |
| 22-26 | Reading instruction booklet quickly, nodding approvingly | Thoroughness: Complete package |
| 26-30 | Text overlay: "Tipp: Nimm 2 für kostenlosen Versand!" | Logic: Shipping hack upsell |
| 30-34 | Holding everything spread out on table, satisfied smile | Value: Full reveal |
| 34-38 | Leaning into camera conspiratorially, pointing down | CTA: Sharing "insider" tip |
