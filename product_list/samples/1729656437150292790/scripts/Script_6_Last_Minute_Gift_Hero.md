## Scripts

**Structure: Last-Minute Gift Hero (Non-Valentine)**

1. **Hook (0-3s):** "No gift idea?" panic → relief
2. **Problem (3-10s):** Gift-giving stress (birthday/anniversary)
3. **Solution (10-25s):** Premium packaging + ready-to-give + affordable
4. **Proof (25-35s):** Looks expensive but budget-friendly
5. **CTA (35-40s):** Save yourself, order now
