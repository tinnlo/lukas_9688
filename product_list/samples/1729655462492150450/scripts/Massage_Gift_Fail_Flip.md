## Scripts

**Concept**: "Gift Fail Flip" - story arc where initially "bad" gift becomes favorite. Targets last-minute gift shoppers. Emotional journey from embarrassment to validation.

### Structure (30s)
- **Hook (0-3s):** Gift fail confession
- **Story (3-20s):** Initial reaction vs. later appreciation
- **Revelation (20-26s):** Now they use it daily
- **CTA (26-30s):** Universal gift positioning

### Visual Strategy

| Seconds | Visual | Purpose |
|:--------|:-------|:--------|
| 00-03 | Person cringing, text: "Ich habe das schlimmste Geschenk gemacht..." | CONFESSION HOOK |
| 03-06 | Flashback: giving wrapped device, recipient's polite but confused smile | AWKWARD MOMENT |
| 06-10 | Text: "Ein Massagegerät? Wirklich?" thought bubble over recipient | IMPLIED DISAPPOINTMENT |
| 10-13 | One week later: recipient using device, "Ahh" expression | ATTITUDE CHANGE |
| 13-16 | Recipient texting: "Das ist das beste Geschenk ever!" message on screen | VALIDATION |
| 16-19 | Montage: Mom using on neck, Dad on shoulders, Grandma on back | UNIVERSAL APPEAL |
| 19-22 | Family fighting over who gets to use it next (humorous) | POPULARITY proof |
| 22-26 | Creator smug face: "Ich hab's gewusst!" | REDEMPTION ARC complete |
| 26-30 | Text: "Das Geschenk, das alle lieben" + product + cart | GIFTING CTA |
