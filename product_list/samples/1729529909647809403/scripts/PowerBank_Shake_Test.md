## Scripts

**Concept**: The "Stress Test" - directly address the #1 fear ("will it fall off?") through aggressive physical demonstration. Target skeptical buyers who've been burned by weak magnetic products. Build trust through extreme proof.

### Structure (25s)

- **Hook (0-3s):** Challenge setup - "Does it really hold?"
- **Test (3-18s):** Escalating shake test sequence
- **Proof (18-21s):** Still charging confirmation
- **CTA (21-25s):** Test passed, trust earned

### Visual Strategy

| Seconds | Visual | Purpose |
|:--------|:-------|:--------|
| 00-03 | Creator with skeptical face, holding phone + power bank, text: "Mein Freund meinte, das fällt ab" | RELATABLE DOUBT - voice viewer's concern |
| 03-06 | Magnetic snap attachment (close-up, crisp audio), green charging icon appears | BASELINE PROOF - it works initially |
| 06-10 | SHAKE TEST LEVEL 1: Gentle shake, no fall | ESCALATION starts - building tension |
| 10-13 | SHAKE TEST LEVEL 2: Aggressive vertical shake over soft bed (safety net visible) | EXTREME PROOF - most products fail here |
| 13-16 | SHAKE TEST LEVEL 3: Spinning phone horizontally like a steering wheel with power bank attached | ABSURD DEMO - entertainment + ultimate proof |
| 16-18 | Close-up: power bank STILL attached, magnetic connection visible | VICTORY MOMENT - defied expectations |
| 18-21 | Screen showing charging icon still active, battery percentage increased during test | FUNCTIONAL CONTINUITY - it didn't just hold, it kept charging |
| 21-25 | Creator's impressed face, thumbs up, text: "Test bestanden!" + finger pointing to cart | TRUST EARNED - social proof through creator endorsement |
