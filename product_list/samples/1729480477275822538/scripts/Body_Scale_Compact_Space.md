## Scripts

Size-focused angle targeting students and small apartment dwellers (Berlin, Munich markets). Addresses two critical objections per image analysis: 1) Small 26cm size (reframe as benefit), 2) App subscription fear (pre-empt with "free" emphasis). Turns potential negatives into selling points.

### Structure (33s)
- Hook: Small apartment problem (0-3s)
- Solution: Compact size demo (3-10s)
- Feature: Smart capabilities (10-20s)
- Objection Handling: Free app clarification (20-28s)
- CTA: Perfect for small spaces (28-33s)

### Visual Strategy
| Seconds | Visual | Purpose |
|:--------|:-------|:--------|
| 00-03 | Cramped bathroom POV, big old scale blocking door | Hook: Relatable pain point for renters |
| 03-06 | Pulling out slim black scale (26x26cm), placing next to laptop for size comparison | Solution: Visual proof of compactness |
| 06-10 | Sliding scale easily under bathroom cabinet / between toilet and wall | Benefit: Space-saving demonstration |
| 10-14 | Feet stepping on scale (barefoot, normal size feet fitting perfectly) | Reassurance: Addresses "too small?" concern |
| 14-17 | LED display lighting up showing weight + temperature | Feature: Extra utility (temperature bonus) |
| 17-20 | Phone screen showing OKOK app with colorful metrics | Tech appeal: Smart features |
| 20-24 | Close-up of app showing "$120 subscription" popup with big red X overlay | Objection handling: Address fear directly |
| 24-28 | Finger tapping "Skip" button, showing free features still work | Proof: No payment required |
| 28-33 | Holding scale in one hand (showing lightness), other hand pointing to link | Portability + CTA |
