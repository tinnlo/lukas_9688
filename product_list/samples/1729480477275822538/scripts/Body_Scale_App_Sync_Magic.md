## Scripts

The "App Sync Demo" format - highest converting angle per video synthesis analysis. Shows the "magic moment" when phone screen updates in real-time with body composition data. This builds trust through visible proof that the smart features actually work, not just marketing claims.

### Structure (35s)
- Hook: Price-value shock opener (0-3s)
- Setup: Stepping on scale (3-6s)
- Magic Moment: Real-time app sync reveal (6-18s)
- Deep Dive: Scrolling through 13+ metrics (18-28s)
- CTA: Price reminder + link (28-35s)

### Visual Strategy
| Seconds | Visual | Purpose |
|:--------|:-------|:--------|
| 00-03 | Split screen: High-tech app interface vs "super günstig" text overlay | Hook: Price-value shock triggers bargain hunter instinct |
| 03-06 | POV feet stepping on black glass scale, LED display lights up | Action: Immediate utility - viewer sees function instantly |
| 06-10 | Close-up: Phone screen showing Bluetooth connection icon + numbers appearing | Trust: Real-time sync proves it's not fake |
| 10-14 | Phone screen showing weight "63.20 kg" matching scale display exactly | Proof: Decimal precision builds credibility |
| 14-18 | Zoom to app showing colorful body composition bars (Fat, Muscle, Water) | Feature reveal: Data richness = "high tech" feel |
| 18-22 | Scrolling through app metrics with finger swipe (BMI, Visceral Fat, Bone Mass) | Education: Show depth of features |
| 22-25 | Red bar warning indicator on screen (e.g., High Body Fat) | Vulnerability: Honest data builds trust |
| 25-28 | Return to full body shot holding phone, satisfied expression | Lifestyle: Relatable usage |
| 28-35 | Holding scale to camera with "super günstig" sticker overlay, pointing down | Conversion: Value emphasis + clear CTA |
