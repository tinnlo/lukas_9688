## Scripts

**Concept: The Mood Switcher (Identity Flexibility)**

Du bist nicht eine Person. Du bist der sympathische Kollege UND der selbstbewusste Macher. Nutzt schnelle Personality-Cuts und POV-Perspektive. Zielgruppe: selbstbewusste Männer (22-32), die Selbstoptimierung als Lifestyle sehen.

### Structure (38s)

- **Hook:** "Welche Version von dir zeigt sich heute?" (0-3s)
- **Personality A:** "The Fresh Guy" mit Azure (3-12s)
- **Personality B:** "The Boss" mit Invincible (12-21s)
- **The Insight:** Beide Seiten sind echt, beide brauchen den richtigen Duft (21-28s)
- **Bundle Logic:** Zwei Identitäten, zwei Düfte, ein Set (28-33s)
- **CTA:** "Werde komplett" (33-38s)

### Visual Strategy

| Seconds | Visual | Purpose |
|:--------|:-------|:--------|
| 00-03 | POV: Spiegel-Shot, creator blickt in Kamera, beide Flaschen auf Waschbecken | Hook: Identity-Frage provozieren |
| 03-06 | Schnitt: Helle Kleidung (weißes T-Shirt/Hemd), Azure Coastal in Hand | Persona 1: Der zugängliche Typ |
| 06-09 | Azure sprühen, lächeln, lockere Körpersprache, Sonnenlicht | Vibe: Approachable, friendly |
| 09-12 | B-Roll: Fahrrad fahren/Kaffee trinken/mit Freunden lachen, Text: "Sympathisch" | Context: Alltags-Sympathie |
| 12-15 | **Hard Cut:** Dunkle Kleidung (schwarzes Hemd/Anzugjacke), The Invincible in Hand | Persona 2: Der dominante Boss |
| 15-18 | Invincible sprühen, ernster Blick, aufrechte Haltung, gedämpftes Licht | Vibe: Commanding, powerful |
| 18-21 | B-Roll: Autoschlüssel greifen/ernster Gang/Blickkontakt halten, Text: "Unvergesslich" | Context: Dominanz-Ausstrahlung |
| 21-24 | Zurück zu Spiegel-Shot, beide Flaschen wieder sichtbar, creator nachdenklich | Reflection: Beide Seiten sind real |
| 24-28 | Beide Flaschen gleichzeitig halten (eine in jeder Hand), Slow-Mo | Unity: Dualität als Stärke |
| 28-31 | Close-up beider Labels: "Azure Coastal" + "The Invincible" nebeneinander | Branding: Produkt-Klarheit |
| 31-33 | Text-Overlay: "2 Identitäten. 1 Set." über beiden Flaschen | Value: Philosophie verkaufen |
| 33-38 | Creator sprüht BEIDE nacheinander (erst Azure links, dann Invincible rechts), Grinsen | CTA: Vollständigkeit als Lösung |
