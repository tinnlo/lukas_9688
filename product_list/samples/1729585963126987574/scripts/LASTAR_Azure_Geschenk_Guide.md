## Scripts

POV einer Freundin, die ein tolles Geschenk gefunden hat. Nutzt die "Partner Validation"-Strategie aus der Video-Synthese (Hook #3, #18). Zielgruppe: Frauen (18-35), die einen erschwinglichen aber hochwertigen Duft für Partner/Bruder/Vater suchen. Positioniert LASTAR als "looks expensive, costs little".

### Structure (45s)
- **Hook:** "Mädels, stop buying him socks" - Problem Statement (0-4s)
- **Solution Reveal:** LASTAR Box zeigen, "Der Geheimtipp" (4-10s)
- **Unboxing:** Hochwertige Verpackung präsentieren, Geschenk-Ästhetik (10-18s)
- **The Smell Test:** Sie lässt IHN riechen, seine Reaktion filmen (18-28s)
- **Value Proposition:** "Riecht teuer, kostet wenig" (28-35s)
- **CTA:** Perfekt für Valentinstag/Geburtstag, Link zum Shop (35-45s)

### Visual Strategy

| Seconds | Visual | Purpose |
|:--------|:-------|:--------|
| 00-02 | POV: Kamera filmt Socken auf Tisch, Hand schiebt sie weg | Hook: Relatability "schlechte Geschenke" |
| 02-04 | Text-Overlay: "Stop. Ich zeige dir was Besseres." | Promise: Lösung kommt |
| 04-07 | LASTAR Box in der Hand (weibliche Hände, schöne Nägel) | Social Proof: Frauen kaufen für Männer |
| 07-10 | Close-up: Box drehen, Logo zeigen, blaues Farbschema | Trust: Hochwertige Verpackung |
| 10-13 | Unboxing: Zellophan vorsichtig entfernen (ASMR) | Gifting: Erlebnis-Faktor |
| 13-16 | Flasche rausholen, gegen Licht halten (blaue Flüssigkeit glänzt) | Aesthetics: "Das schenke ich gerne" |
| 16-18 | Optional: Geschenkschleife oder -papier dazu legen | Context: Ready-to-gift |
| 18-22 | **CUT:** Er (Partner) riecht daran, Augen werden groß, "Wow!" | Proof: Männer lieben es (echte Reaktion) |
| 22-25 | Close-up: Sie lächelt stolz in die Kamera | Emotion: "Ich habe gut gewählt" |
| 25-30 | Split-Screen: Markenparfum (verpixelt) vs LASTAR | Value: Preis-Vergleich ohne genaue € |
| 30-33 | Text: "Er denkt, du hast viel ausgegeben" | Payoff: High perceived value |
| 33-38 | Flasche auf schöner Oberfläche (Marmor/Holz), Lifestyle-Shot | Aspiration: Luxus-Vibe |
| 38-42 | Hand greift zum Handy, TikTok Shop öffnen (Screen-Recording) | Action: Kaufprozess zeigen |
| 42-45 | Gelber Korb-Button, Finger tippt, Text "Jetzt sichern" | CTA: Direkter Call-to-Action |
