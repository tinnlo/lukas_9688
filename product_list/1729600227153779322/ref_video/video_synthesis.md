**Color & Lighting:**
- Bright indoor lighting (home environment)
- Warm tones to convey "Gemütlichkeit" (coziness)
- High contrast on beige/gray tree against neutral backgrounds

## Creative Production Patterns | 创意制作模式

### Common Production Techniques

**Camera Work:**
- **Wide establishing shots:** Show full tree height
- **Close-ups:** Focus on materials (plush, sisal rope texture)
- **Action tracking:** Following a cat climbing (if available)
- **Shake test:** Hand visibly pushing/shaking the tree

**Editing:**
- **Fast cuts:** Every 2-4 seconds to match short attention spans
- **Beat-synced transitions:** Cuts align with music beats
- **Text overlay timing:** Keywords appear exactly when feature is shown

**Lighting:**
- Natural window light (80% of videos)
- Supplemental ring light for close-ups (20%)
- Avoids harsh shadows on fabric

### Production Budget Assessment

**Low-Cost UGC Style (Videos 1, 2, 3):**

## Recommendations for Script Generation | 脚本生成建议

### DO's (必须做)
✅ Lead with **price (€49)** or **stability** in first 3 seconds
✅ Show, don't tell - Use visual demonstration over long descriptions
✅ Include German text overlays for key features
✅ Keep duration **15-25s** for optimal retention
✅ Use "Katzeneltern" (cat parents) to address audience directly
✅ Feature seasonal urgency ("Weihnachten", "Nur heute")
✅ Highlight washability for hygiene-conscious buyers

### DON'Ts (禁止做)
❌ Don't use complex German sentences - Keep it conversational
❌ Don't overload with features - Pick 3 max (stability, size, price)
❌ Don't use stock footage - Real product shots convert better
❌ Don't skip the shake test - Stability is deal-maker
❌ Don't ignore large cat owners - Explicitly say "für große Katzen"

### Visual Requirements
- **Opening shot:** Full tree in frame (establish scale)
- **Mid-section:** Shake test + feature close-ups (posts, caves, hammock)
- **Closing shot:** Cat using tree (if available) OR price graphic

### Voiceover Style (if used)
- **Tone:** Warm, enthusiastic but NOT over-the-top (German preference for understated)
- **Pace:** Moderate (not rushed, but energetic)
- **Language:** Conversational German, avoid English except brand name
- **TTS Option:** ElevenLabs v3 with `[warm]` or `[bright]` cues
