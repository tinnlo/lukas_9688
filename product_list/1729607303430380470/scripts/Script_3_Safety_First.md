# Script 3: Safety First (Social Proof)

## Visual Storyboard

| Time | Visual Action |
|:-----|:--------------|
| **00:00-00:03** | **[HOOK]** Hand holds a cheap, yellowing, old adapter. Aggressively throws it into a trash bin. *Bang sound effect.* |
| **00:03-00:08** | **[AGITATION]** Person shakes head. Text overlay: **"GEFÄHRLICH" (DANGEROUS)**. |
| **00:08-00:15** | **[SOLUTION]** Cut to the **Multi-Function Extender**. Hand taps the robust casing. Camera zooms in on the "CE" mark (if visible) or sturdy build. |
| **00:15-00:22** | **[KEY BENEFIT]** Showing the individual switches. Voiceover emphasizes "Strom sparen" (Saving energy). Click, click, click sounds. |
| **00:22-00:28** | **[GIFT/USE CASE]** Shows it wrapping up or being plugged in a kid's room (safe night light). |
| **00:28-00:35** | **[CTA]** Host holds the product close to camera. "Bestseller" sticker overlay. Points down. |
