## Scripts

Kernidee: Set-Inhalt „explodieren“ lassen → „was ist wofür“ → schnelle Demo → CTA.

### Structure (40s)
- Hook: Tisch voller Zubehör (0–3s)
- Proof: „Alles dabei“ + kurzer Use-Case (3–10s)
- Demo: Padwechsel + 1 Szene am Auto (10–28s)
- Payoff: Vorher/Nachher + „Makita kompatibel“ (28–36s)
- CTA: „Link im Profil“ (36–40s)

### Visual Strategy
| Seconds | Visual | Purpose |
|:--|:--|:--|
| 00–03 | Overhead: Set auf Tisch | Scroll-stop |
| 03–10 | Hände zeigen Pads/Platte/Handschuhe | Value clarity |
| 10–18 | Padwechsel Klick | Ease-of-use |
| 18–28 | 1 Spot am Auto polieren | Proof |
| 28–36 | Before/After + close-up tool | Payoff |
| 36–40 | CTA Text + Packaging/BAG | Conversion |
