## Cross-Product Visual Insights

### Common Success Patterns
1. **Close-Up Texture Shots:** All products benefit from macro tactile shots
2. **Functionality Demonstration:** Show it working, not just sitting
3. **Contrast/Before-After:** Problem ‚Üí Solution visual storytelling
4. **Scale/Context:** Show product in real environment for size reference
5. **ASMR Elements:** Satisfying clicks, swipes, textures appeal to TikTok

### Color Psychology
- **Alien Lamp:** Dark environments + RGB = mystery/cool factor
- **Blanket:** Warm golden tones = comfort/safety
- **Power Socket:** Blue LEDs = tech/modern/trust
- **Pet House:** Neutral + pastels = calm/care

### Recommended Shooting Equipment
- **Lighting:** RGB strip lights for tech products, warm key light for comfort items
