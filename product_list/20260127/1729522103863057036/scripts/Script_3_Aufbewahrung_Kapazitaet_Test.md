## Scripts

Skeptic / capacity test angle: starts with doubt, then proves the product with a real packing challenge. "Does it REALLY fit?" — addresses the #1 objection for storage products. The transformation from doubt to proof is inherently satisfying content.

### Structure (41s)
- Hook: Huge pile of clothes + skeptical claim (0-3s)
- Challenge: "Let's see if this really works" (3-8s)
- Packing demo: Item by item, counting (8-20s)
- Zipper test: Can it close? (20-24s)
- Quality proof: Handle lift, stack test (24-32s)
- Result + CTA: Pile gone, order now (32-41s)
